# Shamu Hardware & Building Supplies Website

A professional website for Shamu Hardware & Building Supplies, located in Chitungwiza, Zimbabwe.

## Features

- **Responsive Design**: Optimized for desktop and mobile devices
- **SEO Optimized**: Includes meta tags, structured data, and canonical URLs
- **Fast Loading**: Clean HTML, CSS, and JavaScript structure
- **Contact Integration**: Phone, WhatsApp, and email links
- **Location Information**: Hours and location details
- **Customer Testimonials**: Social proof section
- **Sticky WhatsApp Button**: For easy contact

## File Structure

- `index.html` - Main HTML file
- `style.css` - Stylesheet
- `script.js` - JavaScript for interactions
- `shamu_logo.jpeg` - Logo image (not included)

## Local Development

To run locally:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Deployment

This site can be deployed to:
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service
